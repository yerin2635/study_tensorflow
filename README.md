# study_tensorflow

學習Tensorflow 的各項紀錄

## Environment

```
Python 3.9.12
tensorflow 2.1.0
cuda 10.1
```

## Local development

1. 建立Python虛擬環境

```shell
$ pip install virtualenv 
$ virtualenv venv     
```

2. 啟動虛擬環境
  
```shell
$ venv\Scripts\activate
```

3. 安裝套件

```shell
$ pip install -r requirements.txt
```
